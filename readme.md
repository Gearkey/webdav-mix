# Webdav-mix：混合 Webdav 客户端

一个基础的 Webdav 客户端，可以将多个储存空间组为资源池，混合模式：

1. 文件结构储存于主储存（disk-0），实际文件根据剩余空间/流量存放在分储存（disk-x）
2. 所有文件储存于主储存，但使用共享空间的分储存代理上传

资源池配置字段：

```
[pool]
mix_type = 1 #混合模式，disk-0 分别作为 0-无混合、1-目录结构、2-文件总储存
target_disk = disk-1 #混合模式中实际上传文件的目标磁盘名称

[disk-0]
hostname = https://dav.jianguoyun.com/dav/ #目标地址
username = xxx@qq.com #用户名
password = 123456 #密码
```
